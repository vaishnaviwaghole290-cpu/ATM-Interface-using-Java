# 💳 ATM Interface using Java

Description:
This project is a simple ATM Interface developed using Java.
It simulates basic banking operations like login, deposit, withdrawal, and balance checking.

Features:
- User Authentication (User ID & PIN)
- Deposit Money
- Withdraw Money
- Check Account Balance

Technologies Used:
- Java
- OOP Concepts

Installation:
1. Install Java (JDK)
2. Open project in any IDE
3. Run the main file
